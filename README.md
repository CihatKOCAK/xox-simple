# XOX Game for Practicum

a simple xox game

## Run it on your computer

Clone the project

```bash
  git clone https://github.com/CihatKOCAK/xox-simple.git
```

Go to client directory and run

    ```bash
    cd xox-simple/client
    npm install
    npm start
    ```

or Yarn

    ```bash
    cd xox-simple/client
    yarn install
    yarn start
    ```
